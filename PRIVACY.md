# Privacy Policy

Glucose A1C Buddy respects your privacy and is designed to minimize data collection.

## Data Access

The app connects to the Dexcom API only with explicit user authorization in order to retrieve continuous glucose monitoring (CGM) data.

This data is used solely to calculate estimated A1C (GMI) values and related summaries.

## Data Storage

- Authentication sessions are stored securely on the backend.
- Access tokens are never embedded in the mobile app bundle.
- No CGM data is permanently stored beyond what is required to fulfill API requests.

## Data Sharing

No user data is sold, shared, or transmitted to third parties.

## Analytics & Tracking

The app does not include advertising trackers.  
Optional error logging may be used to improve app stability.

## User Control

Users may revoke Dexcom access at any time through their Dexcom account settings.

## Changes

This privacy policy may be updated as the application evolves.
